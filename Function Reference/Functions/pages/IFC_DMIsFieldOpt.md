# IFC_DMIsFieldOpt

## Description
Checks if the indicated field, from current IFC Data Mapping,  is optional.

```pascal
FUNCTION IFC_DMIsFieldOpt(
				inStrObjName   : STRING;
				inStrEntryName : STRING;
				inStrFieldName : STRING) : BOOLEAN;
```
