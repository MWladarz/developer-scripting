# PenColorByClass

## Description
Sets the default document pen colors to the pen colors of the active class.&lt;BR&gt;
&lt;BR&gt;
&lt;BR&gt;


```pascal
PROCEDURE PenColorByClass;
```

```python

def vs.PenColorByClass():
    return None
```
