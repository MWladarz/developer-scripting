# vsoInsertParamWidget

## Description
Sets a widget to appear in the Object Info Palette.

```pascal
FUNCTION vsoInsertParamWidget(
				position    : LONGINT;
				parameterID : LONGINT;
				text        : STRING;
				data        : LONGINT) : BOOLEAN;
```
