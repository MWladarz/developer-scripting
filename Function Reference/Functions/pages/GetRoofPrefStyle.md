# GetRoofPrefStyle

## Description
Gets the Roof Style of the Roof Preferences.

```pascal
FUNCTION GetRoofPrefStyle : LONGINT;
```

```python

def vs.GetRoofPrefStyle():
    return LONGINT
```

## Returns
The ref number of the Roof Style of the Roof Preferences. 0 for unstyled.
