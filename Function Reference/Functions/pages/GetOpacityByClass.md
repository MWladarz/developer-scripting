# GetOpacityByClass

## Description
Returns whether the object is using the class opacity.

```pascal
PROCEDURE GetOpacityByClass(
				h             : HANDLE;
				VAR isByClass : BOOLEAN);
```

```python

def vs.GetOpacityByClass(h):
    return isByClass
```

|Name|Type|Description|

|h|HANDLE||
|isByClass|BOOLEAN||
