# vsoAppendWidget

## Description
Sets a widget to appear at the end of the widgets in the Object Info Palette.

```pascal
FUNCTION vsoAppendWidget(
				widgetType : LONGINT;
				mappingID  : LONGINT;
				text       : STRING;
				data       : LONGINT) : BOOLEAN;
```

def vs.vsoAppendWidget(widgetType, mappingID, text, data):
    return BOOLEAN
