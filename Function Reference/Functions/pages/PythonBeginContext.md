# PythonBeginContext

## Description
This function creates a context in which PythonExecute scripts are run.&lt;BR&gt;
&lt;BR&gt;
This function consecutive python scripts to be executed inside the same python environment.

```pascal
PROCEDURE PythonBeginContext;
```

    return None
