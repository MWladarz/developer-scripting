# ActiveClass

## Description
Returns the name of the active class of the document.

```pascal
FUNCTION ActiveClass : STRING;
```

```python

def vs.ActiveClass():
    return STRING
```

## Examples
```pascal
activeClName:= ActiveClass;
```
