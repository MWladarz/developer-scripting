# DTM6_IsTypeVisible

## Description
Return true if the specified DTM type is visible of the passed DTM object.

```pascal
FUNCTION DTM6_IsTypeVisible(
				hDTMObject : HANDLE;
				TINType    : INTEGER) : BOOLEAN;
```

    return BOOLEAN
