# IFC_DMIsFieldEmpty

## Description
Checks if the indicated field, from current IFC Data Mapping, is empty.

```pascal
FUNCTION IFC_DMIsFieldEmpty(
				inStrObjName   : STRING;
				inStrEntryName : STRING;
				inStrFieldName : STRING) : BOOLEAN;
```
