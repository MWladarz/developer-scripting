# IFC_DMIsFieldEnabled

## Description
Checks if the indicated field, from current IFC Data Mapping, is enabled.

```pascal
FUNCTION IFC_DMIsFieldEnabled(
				inStrObjName   : STRING;
				inStrEntryName : STRING;
				inStrFieldName : STRING) : BOOLEAN;
```
