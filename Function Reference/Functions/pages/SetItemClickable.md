# SetItemClickable

## Description
Sets the specified item to generate events when clicked.  Currently only static text and images are supported.

```pascal
PROCEDURE SetItemClickable(
				dialogID    : LONGINT;
				componentID : LONGINT;
				clickable   : BOOLEAN);
```
