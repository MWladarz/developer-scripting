# GetSlabStyle

## Description
Gets the Slab Style of a slab.

```pascal
FUNCTION GetSlabStyle(slab : HANDLE) : LONGINT;
```

```python

def vs.GetSlabStyle(slab):
    return LONGINT
```

## Parameters
|Name|Type|Description|
|---|---|---|
|slab|HANDLE|The slab.|

## Returns
The ref number of the Slab Style of the slab. 0 for unstyled.
