# SheetList

## Description
Returns the name of the sheet specified by index.

```pascal
FUNCTION SheetList(sheetIndex : INTEGER) : STRING;
```

```python

def vs.SheetList(sheetIndex):
    return STRING
```

## Parameters
|Name|Type|Description|
|---|---|---|
|sheetIndex|INTEGER|Index of the sheet|

## Returns
Returns a string containing the name of the sheet.
