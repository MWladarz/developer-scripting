# Space

## Description
Procedure Space writes a space to the current output file.

```pascal
PROCEDURE Space(n : INTEGER);
```

```python

    return None
```

## Parameters
|Name|Type|Description|
|---|---|---|
|n|INTEGER|Number of spaces.|
