# SetImageCropVisible

## Description
Set the image crop visibility.

```pascal
PROCEDURE SetImageCropVisible(
				object    : HANDLE;
				isVisible : BOOLEAN);
```

## Parameters
|Name|Type|Description|

|object|HANDLE||
|isVisible|BOOLEAN|If TRUE the crop become visible, if FALSE - invisible.|
